# pmux

pmux is a programmable multiplexer that takes a simple configuration of commands you want run and executes them in a tmux session.

## Getting Started

Install pmux with:

```
npm install -g pmux
```

Execute pmux with:

```
pmux <path-to-configuration-file>
```
